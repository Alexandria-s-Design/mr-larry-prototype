# Life Dreams Wealth — 360° Interactive eLearning

A comprehensive showcase of 16 eLearning interaction types overlaid on a draggable 360° panorama of the Manhattan financial district. Built as a single HTML file with zero dependencies.

## Live Demo
[View on GitHub Pages](https://charlesmartinedd.github.io/mr-larry-prototype/)

## Brand Identity
- **Colors**: Gold (#D4AF37) / Navy (#0F172A) / White — luxury financial feel
- **Logo**: Diamond gem mark with "LDW" monogram
- **Tagline**: Financial Literacy — 360° Interactive

## Interaction Types

| Category | Color | Interactions |
|----------|-------|-------------|
| **Content Display** | Gold | Info Panel, Image Gallery, Video Embed, Audio Player, PDF Viewer |
| **Assessment** | Coral | Multiple Choice, Multiple Select, True/False, Drag & Drop, Fill in the Blank, Likert Survey |
| **Interactive Tools** | Teal | Compound Interest Calculator, Timeline, Flashcards |
| **Gamification** | Purple | Scenario Branching, Timed Challenge |

## Features
- Draggable 360° panorama (Lower Manhattan Financial District)
- 16 color-coded interactive hotspots
- Progress tracking HUD (0/16 → 16/16)
- Fully responsive overlay panels
- Gold/navy luxury brand identity
- Zero external dependencies — single HTML file

## Part of Life Dreams Wealth
This 360° prototype is the web-based companion to the **Life Dreams Wealth 3D Museum** — a Godot-powered first-person walkthrough experience covering credit, investing, budgeting, debt management, retirement, and tax fundamentals.
