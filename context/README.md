# Life Wealth Dreams — Project Context & Historical Assets

This folder contains all historical context, design assets, and prior work for the Life Wealth Dreams financial literacy curriculum project (originally "Mr. Larry's Financial Literacy"). These files were sourced from Google Drive and represent the foundation of wants, needs, and prior development.

## Folder Structure

### `documents/`
- **Larry Pricing.docx** — Pricing model and structure for the curriculum
- **Simulation Branches - Nutrition Modules.xlsx** — Branching logic and decision trees for the nutrition simulation modules

### `presentations/`
- **Life [Autosaved].pptx** — Main Life Dreams presentation deck (~1.1MB)
- **Life.pptx** — Additional presentation from the nutrition module pics folder
- **life-autosaved-captivate/** — Adobe Captivate project export (XML, attachments, playlists)
- **presentation1-captivate/** — Secondary Captivate project export

### `logos/`
- 19 logo variations and brand assets including:
  - `Head.png` / `Head 2.png` — Character head/avatar designs
  - `Logo1-3.png`, `logo9.png` — Logo iterations
  - `My Post 4-11.png` — Social media post designs
  - `Font.png` — Typography reference
  - `IceBerg1.png` / `Iceberg2.png` — Iceberg model concept graphics

### `nutrition-modules/`
- **simulation-published/** — Published Captivate nutrition simulation (index.html + data/)
- **nutrition-extract/** — Extracted nutrition simulation variant
- **nutrition-food-extract/** — Food-focused nutrition simulation variant
- **Coupon Pic.png/.jpg** — Coupon visual used in nutrition module
- **my-visual_47335948.png** — Custom visual for nutrition content
- All three simulation folders contain: player.js, fonts (.woff), images, audio (snd0.mp3), and browsersupport.js

### `vr-simulation/`
- Complete SCORM 1.2-packaged VR simulation built with Adobe Captivate
- **index_scorm.html** — Main entry point
- **scormdriver.js** — SCORM communication layer
- **dr/** — 360-degree image data (img1.json = panoramic image data)
- **assets/** — CSS, JS, HTML images, TOC icons
- **imsmanifest.xml** — SCORM manifest
- **project.txt** — Project configuration

### `scorm-packages/`
- **Nutrition 1 Simulation Demo 1.Inc..zip** — Packaged SCORM nutrition simulation
- **spcd495022872dde3ea1d873f_SCORM1.2_captivate.zip** — Captivate SCORM export

### `meetings-ndas/`
- **Meeting_Transcript.docx** — Transcript from July 2022 meeting with Amanda
- **Amanda.NDA.pdf** — NDA with Amanda
- **Amanda.Alexandrias_Design.NDA.pdf** — Alexandria's Design NDA with Amanda
- **Bonita.NDA.pdf** — NDA with Bonita (Alexandria's Design)
- Note: Two meeting recording MP4s (~870MB total) remain in Google Drive due to size

### `website-assets/`
- **stock-photos/** — 70+ Unsplash stock photographs used for website/marketing (nature, ice, wildlife themes)
- **iceberg-concept/** — Iceberg model imagery (metaphor for hidden financial knowledge)
- **Powtoons Header.mp4** — Animated header video (~52MB)

## Timeline
- **2020** — Initial development: presentations, nutrition simulations, SCORM packages, logos
- **2022** — Business development meetings with Amanda, NDAs executed
- **2025-2026** — Revival as "Life Wealth Dreams" interactive eLearning prototype
